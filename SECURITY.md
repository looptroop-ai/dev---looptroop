# Security Policy

LoopTroop executes AI-generated coding work in local repositories and isolated worktrees. Please report security issues involving command execution, workspace isolation, secret exposure, prompt injection, unsafe PR automation, dependency handling, or durable artifact leakage.

Do not disclose vulnerabilities publicly until maintainers have reviewed them. Prefer GitHub private vulnerability reporting for this repository when available. If private reporting is unavailable, contact the maintainers through a non-public channel before opening a public issue.

When reporting, include:

- The affected version or commit.
- Reproduction steps and expected impact.
- Any logs, prompts, generated commands, or `.ticket/**` artifacts needed to understand the issue.
- Whether secrets, credentials, or private source code may have been exposed.
